# Eucalipto

Projeto inicial Eucalipto

## 🚀 Começando

* mvn clean install;
* java -jar cadastro-0.0.1-SNAPSHOT.jar;
* yarn build; 
* yarn start;
* heroku git:remote -a eucalipto-cadastro
* git subtree push --prefix backend heroku main


### 📋 Pré-requisitos

* Uma boa máquina de escrever e tempo.
